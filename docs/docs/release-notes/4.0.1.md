# Release 4.0.1 (2026-02-10)

This release captures the work after `v4.0.0`, focusing on devnet tooling, Hydra docs, and a small but important Hydra fee fix.

## Highlights
- Local Hydra devnet setup is now scripted with a cluster reset workflow.
- New local devnet and testnet/mainnet docs make onboarding easier.

## Changes
### Fixes
- Start the transaction fee at 0 for already balanced Hydra transactions to avoid insufficient input errors.

### Devnet tooling
- Added a `reset-cluster.sh` convenience script for local devnet resets.
- Added Cardano node config and genesis files for the devnet cluster.
- Improved credential generation, UTxO verification, and seeding scripts.

### Docs
- Added new Hydra JS client guides for local devnet and testnet/mainnet.
- Refined getting started and installation docs.
- Updated Hydra API reference docs and project README sections.
