# Release 4.0.0 (2026-02-10)

This release finalizes the Hydra integration from `v4.0.0-rc1` and focuses on documentation polish, navigation fixes, and a small API fix for `kuber-hydra`.

## Highlights
- `kuber-hydra` is included as a first-class module for Hydra head commands, queries, and transaction building.
- Docs were reorganized and updated to make Hydra onboarding and the tx builder reference easier to follow.

## Changes
### Included from v4.0.0-rc1
- New `kuber-hydra` module with command endpoints under `/api/hydra` and query endpoints under `/api/hydra/query`.
- Hydra transaction builder for input selection, optional signing keys, and UTxO filtering by txin and address.
- New Hydra documentation and sequence diagrams.

### Updates in v4.0.0
- Added common Kuber functions to the Hydra JS client docs.
- Added `yarn doc:validate` for docs validation.
- Added tx-builder reference to the buildTx docs.
- Improved getting started content and navigation.
- Fixed broken links for `kuber-hydra` and `kuber-client`.
- Updated README and reorganized documentation sections.

## Fixes
- Refactored and fixed decommit API endpoints in `kuber-hydra`.

## Compatibility
- cardano-api: 10.1
- kuber-client: v3.3.5
